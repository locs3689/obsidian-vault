# 第8回：リード・ユーザー法

## イノベーションの民主化

これまでのイノベーションは、「メーカーによるイノベーションと、それを支える社会の仕組み」という構図。特別な人たちの独占的な領域であった。
ところが、最近は消費者自身がものづくりの担い手となり、社会を変え始めている。

## ユーザー・イノベーション

商品の使い手が、その商品の改良・開発に、自ら取り組む現象。
CD → CDジャケット、反射鏡、時計などに…
ユーザー・イノベーションは「おもちゃ」のような形で登場する。

ヨーロッパでは、深刻化する駐車スペースの問題がある。そのため、駐車技術で工夫。
MITが生み出した、ショッピングカートを使ったプロトタイプ（折りたためるという特徴を表現）から、コンパクトカーを提唱。

## リード・ユーザー

- リード・ユーザー
	- 革新を起こすユーザーのこと。
	- 別名「エクストリームユーザー」
	- 利用頻度の低い人や高い人
- リード・ユーザー法
	- そのリードユーザーを探し出して、そのアイデアを学ぶ手法。

### 例）初音ミク

- 音声合成ソフトの2次元キャラクター
	- ソフトウェアのパッケージ用。
	- 販促ツールにすぎない。
- 3,000本売れたら大ヒットの業界で、4万本売れた。

ミクを動かしたいユーザーが出現。最初は少しの動き（ネギを振るだけ）しかなかった。

続いて、これを3次元で動かしたいユーザーが登場。できればもう少し可愛く。
誰でも簡単に、3Dの初音ミクを歌って踊らせることができるダンスソフトを、ユーザーが開発。

技術は一度誕生すると、進化は加速度的。
開発元（メーカー）にも変化が起き、ホログラムで3Dのミクを映してコンサートを実施。

## なぜユーザーイノベーションが起こる？

どうしても欲しいのに、メーカーが作ってくれないから。

- 期待利益仮説
	- あるアイディアを実現させた時の利益が、「ユーザー > メーカー」の時。
	- 例）難病向け医療器具
		- 投資効率を考え、市場規模を求めるメーカー。
		- 自分の生活・生命に直結する患者。
		- アミット・ゴファ
			- イスラエルのエンジニア
			- 交通事故による脊髄損傷で下半身麻痺
			- 車椅子生活
			- ロボットスーツ「ReWalk」の開発
		- メーカーの利益が出るかわからないが、ユーザーは生活の質を上げるために欲しい。
- 情報の粘着性仮説
	- 相手に情報を伝えるのが容易
		- 粘着性が低い
		- メーカーがイノベーション
	- 相手に情報を伝えるのが困難
		- 粘着性が高い
		- ユーザーがイノベーション
		- 例）アスリートの用具開発
			- 一流にしかわからない経験や感性を語る選手
				- フィギュアスケートで「ガリガリではなくシュルシュル」の靴が欲しい。
			- 具体的なスペックを必要とするメーカー
				- メーカーの協力を得て、違いを動画に収め、商品開発。
		- 例）ハスラー
			- 車体に穴を開けて、収納を作っている人がいた。
				- 先に穴を開けておく。
			- 車でシャワーを浴びている人がいた。
				- 小綺麗で、水対策もした。シャワーブースを作る。
		- 例）ワークマンの厨房靴
			- 雨の日に妊婦が履いていて、それがSNSで拡散。
				- デザインを変えて販売。
