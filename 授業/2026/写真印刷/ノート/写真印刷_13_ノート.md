# 第13回

- 商品カタログの印刷は、輪転機のセットが面倒なので？一発で決めたい。
- レタッチ（色校朱書き対象別の修正ランキング）
	1. 商品写真
	2. 色玉・素材
	3. イメージ
	4. モデル
	5. 商品合成
- 2014年には、JapanColorの基準を満たさない会社があった。
	- レタッチ以前に、印刷以前の色変動が問題。
- マットコート紙や非塗工紙では色域が狭くなるが、対応した出力プロファイルで変換することなく印刷していることがある。
- 先生は、カメラが印刷されたパンフレットと各メーカーのHPにあるPDFを比較した。

- レタッチの目的
	- 色を最適に（忠実に、正確に、好ましく）再現する。
	- 階調を最適化する。
	- 印刷方式に適した画像にする。
	- 商品や人物を魅力的に見せる。
	- 印刷時に発生しやすい問題を防ぐ。
		- マゼンタに転ぶと不満を感じる人が多い。
- レタッチ良否の判断者
	- 印刷物を手にした人
		- もっとも無難で、これを重視することが多い。
	- 依頼者
	- 印刷会社の品質責任者
	- 撮影者
- 画像品質の何に目がいくかを主観評価すると、まずは「色・トーン」。
	- 色がついていると嫌。
	- その次に、他のことに目がいく。
-  ニアレストネイバーとバイキュービック

| 比較項目     | ニアレストネイバー法      | バイキュービック法       |
| -------- | --------------- | --------------- |
| 参照する画素数  | 1ピクセル（最も近いもの）   | 16ピクセル（周囲4×4）   |
| 処理速度     | 圧倒的に速い          | 遅い（計算量が多い）      |
| 境界線・輪郭   | カクカクする（ジャギーが出る） | 滑らかで自然          |
| 新しい色の生成  | しない（元の色のみを複製）   | する（計算による中間色を合成） |
| 得意な画像タイプ | ドット絵、QRコード、図形   | 写真、自然画、グラデーション  |
