# 第6回：時間とは？

## 前回までの振り返り

- PiperとLost Thingは別々の回ではなく、同じ問いの連続であった？
- Piperを「動く / 挑戦する / 世界を広げる」作品
  Lost Thingを「見えなくなる / 忘れる / 居場所を探す」作品
- 童心＝善、大人＝悪という単純化を避けて、子ども性と大人性の両方を制作資源とする必要がある？
	- 「大人でしか作れないものがある、あるいは子供でしか〜」のように、二項対立の視点になってしまいやすい点に注意する。

人を楽しませるために、考え続ける必要がある。

「鉤括弧」記事の分析を見ることが、いかにコスパの悪いことか。
作品をつくるのは、あるいは読み解くのはどういうことか。

## Lost Thing 作者の考え

- ガラクタ管理局は、「心が死んだもの」の最たるもの。
- 芸術は誇張することから始まる。
	- 現実世界の要素を取り上げ、ある側面を大きくデフォルメして、人々の注意を引く。
	- 観客には「自分の生活との共通点」を考えてほしい。
- 重要なものは、目の高さよりも下にある。
- 最後の世界について
	- 共通の伝達手段、生態系、性格や習慣について、制作の際に意識。
	- 捨て去られた芸術が爆発。
- Lost Thingとは、生き物のことではないのかも？
- 製作者自身も解釈を続けている。

## 先生の分析

- その人の感情とどう重ねるかは、人それぞれ。
	- 芸術の「懐の深さ」とは、この部分。
- 作り手側が「かもしれない」「思う」という言葉を多用している。
- 最後の別れのシーンは、「失ったのではなく、しまったのではないか」

## みんなの意見

>ロストシングはあの生き物じゃないかもしれないって言ってたけど、何なのかみんなの意見聞きたい

大人になると失うもの？大人になったら見えなくなるもの？


>自分でよくわかんないモノをよくわからないからそのまま表現しているんですか

人間が生きている理由など、みんな分からない。
だがしかし、宗教というものがあり、それを信仰している人がいる。そのように、芸術に触れ合うことで、寄り添ってくれる作品もある。
人間とはよく分からない、というところにタッチしている作品？


> 作者自身、結論を出すことでの作品の単純化を避けている（固定化されない、流動的な作品であろうとしている）

先生はこれを否定。（なぜ？）
