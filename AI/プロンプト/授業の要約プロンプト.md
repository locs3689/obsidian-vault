## 音声

```
以下の[音声データ/文字起こしテキスト]の内容を、要約や省略を一切せずに、可能な限り詳細な文章として書き出してください。

【出力時のルール】
1. 話の文脈が正確に伝わるように、展開に沿って時系列で記述すること。
2. 会話の中に出てきた具体的なエピソード、固有名詞、数値などは絶対に省略せず、すべて含めること。
3. 結論だけでなく、「なぜそのような話になったのか」という思考の過程や理由も詳細に書き出すこと。
4. 全体が読みやすくなるように、話題が変わるタイミングで適切な【見出し】をつけること。
```

```
添付した【音声データ】の内容を基に、情報を一切落とさずに、詳細で読みやすい文章（レポート）を作成してください。
ハルシネーション（情報の捏造や、音声で語られていない推測の追加）を絶対に防ぐため、以下の【ステップ1】【ステップ2】の順に処理を実行し、両方を出力してください。

【ステップ1：音声からの重要要素の網羅的抽出】
文章化する前に、音声データから聞き取れる以下の要素を、箇条書きで「すべて」リストアップしてください。
・話題、テーマの変遷
・登場した固有名詞（人名、企業名、サービス名、場所など）
・具体的なエピソード、具体例、余談、雑談
・数値（金額、年号、日付、時間、数量など）
・発言者がその結論に至った背景や理由（※音声の中で「明確に発言されているもの」に限る。AIによる推測は一切不可）

【ステップ2：詳細な文章の構築】
ステップ1で抽出したすべての要素を確実に組み込み、以下のルールに従って詳細な文章として書き出してください。

［出力時のルール］
1. 話の展開に沿って時系列で記述し、文脈が正確に伝わるようにすること。
2. 話題が変わるタイミングで、内容に沿った適切な【見出し】（## などのマークダウン）をつけること。
3. ステップ1で抽出した固有名詞、数値、具体例、余談などは一切省略せずに、必ず文章内に自然な形で盛り込むこと。
4. 単なる結論の羅列ではなく、「なぜその話になったのか」という会話のプロセスも、音声の事実に基づき記述すること。
```

## 動画

```
以下の[動画データ／動画の文字起こしテキスト]の内容を、要約や省略を一切せずに、可能な限り詳細な文章として書き出してください。

【出力時のルール】
1. 話の文脈が正確に伝わるように、展開に沿って時系列で記述すること。  
2. 会話の中に出てきた具体的なエピソード、固有名詞、数値などは絶対に省略せず、すべて含めること。 
3. 結論だけでなく、「なぜそのような話になったのか」という思考の過程や理由も詳細に書き出すこと。
4. 音声だけでなく、映像内の重要な視覚情報（テロップ、スライドに書かれた文字・図表、実演の手順、登場人物のジェスチャーや表情など）も併せて詳細に記述すること。
5. 会話の中で「これ」「あの」などの指示代名詞が使われている場合は、映像から読み取れる具体的な対象物をカッコ書きなどで補足して記述すること（例：「これ（右側の赤いグラフ）が示すように〜」）。
6. 全体が読みやすくなるように、話題が変わるタイミングや場面が転換するタイミングで適切な【見出し】をつけること。
```
